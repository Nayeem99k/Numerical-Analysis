গাউসিয়ান অপসারণ পদ্ধতি বা সরাসরি পদ্ধতি

থিওরি এবং প্রমাণ

গাউসিয়ান অপসারণ পদ্ধতি সমীকরণ সমাধানের জন্য একটি কার্যকর পদ্ধতি, যা ধাপে ধাপে উপরের ত্রিকোণমাতৃক (Upper Triangular Matrix) গঠনের মাধ্যমে কাজ করে।

Ax = B ... (1)
(1)
এখানে,
A = 
        | a11 a12 ... a1n |
        | a21 a22 ... a2n |
        | ...   ... ... |
        | an1 an2 ... ann |
এবং, x = 
| x1 x2 ... xn |T
এবং B = 
| b1 b2 ... bn |T

প্রথম ধাপ: প্রথম সারির উপাদান ব্যবহার করে অন্যান্য সারি শূন্য করা

R2 - (a21/a11) * R1 → R2
R3 - (a31/a11) * R1 → R3
...
Rn - (an1/a11) * R1 → Rn
(2)
প্রথম সারির প্রথম উপাদানকে (a11) পিভট ধরে নিয়ে অন্যান্য সারির প্রথম উপাদানগুলোকে শূন্য করা হয়। এভাবে নতুন ম্যাট্রিক্স পাওয়া যায়: 
        | a11  a12 ... a1n |  | x1 |   =   | b1  |
        |  0   a22 ... a2n |  | x2 |       | b2' |
        |  0    0  ... a3n |  | x3 |       | b3' |
        | ...   ... ... |  |... |       | ... |
        |  0    0  ... ann |  | xn |       | bn' |

দ্বিতীয় ধাপ: দ্বিতীয় সারির পিভট ব্যবহার

R3 - (a32/a22) * R2 → R3
...
Rn - (an2/a22) * R2 → Rn
(3)

চূড়ান্ত রূপান্তর

Ux = C ... (4)
(4)
যেখানে, U হল উপরের ত্রিকোণমাতৃক এবং C নতুন পরিবর্তিত ধ্রুবক ভেক্টর। এই অবস্থায় Back Substitution পদ্ধতি প্রয়োগ করে x এর মান নির্ণয় করা হয়।

উপসংহার

এভাবে, ধাপে ধাপে পিভট নির্বাচন এবং অন্যান্য উপাদান শূন্য করার মাধ্যমে গাউসিয়ান অপসারণ পদ্ধতি কাজ করে। প্রতিটি ধাপের গাণিতিক বিশ্লেষণ থেকে দেখা যায়, মূল সমীকরণ (1) থেকে (4) পর্যন্ত রূপান্তর সম্পূর্ণ যৌক্তিক এবং সঠিক সমাধান প্রদান করে।
